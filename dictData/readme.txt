the 'DataGen.py' file is made to generate the 'DataDump.py' file.
the 'DataDump.py' file can be edited manually, and is used to generate 3 dictionary files that contain the alphabetic and numeric characters and their corresponding brail code
for three languages. (English, french and arabic)
these files are (enData, frData and arData)
Note : 
to test the arabic language, the language for non unicode programs should be set to arabic (see control panel administrative language)
